# Roadmap

## v0.1 当前版本

- 样例 detection schema。
- 事件聚合、风险评分和巡检报告。
- CLI、FastAPI、Docker、smoke test 和项目文档。

## v0.2 接入真实模型

- 增加 YOLOv8 detector backend。
- 支持图片目录和视频抽帧输入。
- 输出证据帧路径和可视化框图。
- 增加模型权重路径、置信度、类别过滤配置。

## v0.3 巡检业务增强

- 引入航线、禁区、白名单车辆和计划任务。
- 多帧事件聚合升级为 Track ID 级别。
- 支持告警工单、人工复核和误报反馈。
- 增加日报、周报和趋势统计。

## v0.4 生产交付

- GPU 推理服务容器化。
- 异步任务队列处理视频。
- 对象存储保存原始帧、检测图和报告。
- 监控模型延迟、误报率、漏报率和告警确认率。
